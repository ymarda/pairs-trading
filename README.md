# Pairs Trading: Statistical Arbitrage Strategy

A cointegration-based pairs trading system implemented in Python.

## Overview
This project identifies cointegrated pairs of equities, generates mean-reversion trading signals from the spread, and backtests the strategy with realistic transaction costs.

## Status
🚧 Work in progress

## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Project structure
- `notebooks/` — exploratory analysis
- `src/` — reusable modules
- `data/` — input data (not committed)
- `plots/` — output figures
- `tests/` — unit tests
